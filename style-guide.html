To rebrand: swap --accent, --accent-dark, and --accent-light. Everything else stays neutral. The accent color appears on overlines, star ratings, logo highlight, and hover states. The accent-light works for soft highlights, card backgrounds, tags, badges, and subtle emphasis areas.

:root { --black: #000; --white: #fff; --tone: #f7f7f8; --accent: #0872cc; /* ← swap this */ --accent-dark: #0660ab; /* ← swap this */ --accent-light: #c2cdff; /* ← swap this */ --mid: #949698; --dark: #4c4c4b; --light-border: #dfe0e1; }

All buttons: black background (#000), white text, no border-radius, no rounded corners. Hover state: background transitions to --dark (#4c4c4b) over 0.2s. No box-shadow, no gradients. Always flat.

04 / Form Inputs
Underline Inputs (contact forms)

No visible border except a 1px bottom line in --light-border. Focus state: border-color transitions to --black. No background color, no border-radius. Select gets a custom SVG dropdown arrow. Placeholder color: --mid.

Inline Email Form (newsletter)

Input and button share a border. Input: 1px solid --light-border, no right border. Button: black background, white text. No gap between them. Max-width: 400px.

07 / Animation

Two animations in the system. Both are minimal.

Scroll Reveal
.reveal { opacity: 0; transform: translateY(20px); transition: opacity 0.6s ease, transform 0.6s ease; } .reveal.visible { opacity: 1; transform: translateY(0); } /* JS trigger */ const observer = new IntersectionObserver((entries) => { entries.forEach(entry => { if (entry.isIntersecting) entry.target.classList.add('visible'); }); }, { threshold: 0.1 }); document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
Hover Transitions

08 / Responsive Breakpoints
/* Tablet: 900px */ @media (max-width: 900px) { - Horizontal padding: 48px → 24px - Three-col grid → single column - Testimonial grid → single column - Split section → stacked - Contact grid → single column } /* Mobile: 600px */ @media (max-width: 600px) { - Nav links hidden except CTA - Email form → stacked vertically - Contact row (name/email) → single column - Sticky mobile CTA bar appears (fixed bottom) - Footer gets extra bottom padding for CTA clearance }

10 / Rebranding Checklist
To use this system for a different brand: 1. COLORS - Swap --accent (#0872cc) with your brand color - Generate --accent-dark (10-15% darker) for hover states - Generate --accent-light (soft pastel version) for highlights, tags, badges - Everything else (black, white, tone, mid, dark, border) stays the same 2. FONTS - Display: swap Optima for any elegant display font - Body: swap Outfit for any clean sans-serif - Mono: swap DM Mono for any monospace - Keep the letter-spacing rules (-1px headings, -2px hero, 2px overlines) 3. LOGO - Same format: BRAND NAME in mono, uppercase, 12px, 2px tracking - Second word gets --accent color 4. FAVICON - SVG: black square, your initials in --accent, monospace font 5. CONTENT - Hero: one sentence, max 8 words, with italic emphasis - Subtext: one sentence, max 20 words - Three-col: your three core services - Work list: client URLs - Testimonials: 3 client quotes - Contact form: customize dropdown options - Banner: your service keywords with middot separators 6. TECHNICAL - Formspree: swap endpoint URL - GA4: swap measurement ID - Schema markup: update business info - OG image: regenerate with your brand
