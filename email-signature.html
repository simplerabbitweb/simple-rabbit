How to add to Gmail / Google Workspace

  1. Open this file in Chrome
  2. Click anywhere inside the white box below
  3. Press Cmd+A to select all, then Cmd+C to copy
  4. Open Gmail → Settings (gear icon) → See all settings
  5. Go to the General tab → scroll to Signature
  6. Create a new signature → paste with Cmd+V
  7. Click Save Changes at the bottom
